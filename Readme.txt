

---------------------------------- CLI: --------------------------------

---------- Create New Category: ----------
to create a new category you have to type: 
php artisan create:category
Then answer the questions by typing
Note:
Parent is optional.

---------- Create New Product: ----------
to create a new product you have to type: 
php artisan create:product
Then answer the questions by typing
Note:
Image is optional.

---------- Delete a Category----------
to delete a category you have to type: 
php artisan delete:category
Then type the name of Category as it is

---------- Delete a Product ----------
to delete a Product you have to type: 
php artisan delete:product
Then type the name of Product as it is
